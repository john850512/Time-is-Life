## Description
設計與實作一款整合Google Map API導航技術與路徑判斷功能的道路避讓即時警示系統，在車輛駕駛將要行駛時使用APP開啟偵測功能，定時判斷是否有救護車即將經過駕駛人當前所在位置進而發出訊息提醒。

#### Time is Life–道路避讓即時警示系統
